This webpage is designed using HTML, Bootstrap, and jQuery to display the relevance of different types of projects at VIT — EDI, Design Thinking, and Course Projects — using toggleable dynamic tabs/pills.

**HTML (HyperText Markup Language)**

Used to create the basic structure of the webpage including headings, tabs, and content sections.

**Bootstrap**

A popular CSS framework included through a CDN link.
It provides ready-made designs and components.
In this code, Bootstrap is used for:

1. Styling (colors, spacing, layout)

2. Navigation pills (tabs)

3. Responsive design

**jQuery**

A JavaScript library that makes it easier to work with HTML elements.In this code, jQuery is used to add smooth animations when switching between tabs.

**Bootstrap Bundle JS**

Enables the interactive features of Bootstrap (such as clicking between tabs).

**Custom CSS**

Additional styling (background color, card shadow, font style) to make the UI visually attractive.
